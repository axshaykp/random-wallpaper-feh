# random-wallpaper-feh

A bash script to download a random wallpaper from Wallpaper Cave and apply wallpaper using feh.

## Requirements

- feh

## Setup

```sh
git clone https://github.com/axxsh/random-wallpaper-feh.git
cd random-wallpaper-feh
chmod +x ranwall.sh
```

## Usage

```sh
./ranwall
```
